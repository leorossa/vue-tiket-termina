IsDisableEditVisitObject запрещено изменять список
объектов посещения (в заказе могут присутствовать только объекты посещения
с флагом IsRequire)

IsDisableEditVisitor запрещено изменять
количество и категории посетителей (в заказе могут присутствовать только
категории посетителей с полем RequireVisitorCount с предлагаемым кол-вом)

IsVisitObjectUseForCost объекты посещения
используются для расчета стоимости

IsCategoryVisitorUseForCost категории посетителей
используются для расчета стоимости

IsVisitorCountUseForCost кол-во посетителей
использутеся для расчета стоимости

IsUseOneCategory в список посетителей
можно добавить только одну из доступных категорий посетителей

ActiveKindId способ определения
возможного интервала дат, в течение которого можно посетить музей:
1. В течение X дней после оплаты (по умолчанию);
2. В течение X дней после оплаты;
3. В любой день из интервала дат;
4. В любой день из списка дат.

IsNeedVisitDate означает, что при заказе
обязательно указывать дату визита:
ActiveKindId=1 нужно указать любую желаемую дату;
ActiveKindId=2 нужно указать любую желаемую дату;
ActiveKindId=3 нужно указать любую желаемую дату из интервала
dtBegin...dtEnd;
ActiveKindId=4 нужно указать любую желаемую дату из списка

IsNeedVisitTime если true, значит дату
надо указать вместе со временем посещения (время выбирается из сетки
времен)

ActiveDays в течение скольки дней
после оплаты можно посетить музей (если ActiveKindId=1 или 2);

IsRequire объект посещения выбран
по умолчанию (обычно это сопровождается IsDisableEditVisitObject=True)

RequireVisitorCount выбранное по умолчанию
кол-во посетителей данной категории (обычно это сопровождается
IsDisableEditVisitor=True)

VisitObjectId идентификатор объекта
посещения (может отсутствовать, используется для
IsVisitObjectUseForCost=False)


CategoryVisitorId идентификатор категории
посетителей (может отсутствовать, используется для
IsCategoryVisitorUseForCost=False)
